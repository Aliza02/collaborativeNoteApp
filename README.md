# notesapp

A Collaborative Notes App
### How to run Application :
Just clone the repository, open it in an editor, and run the application.
### Explanation
The application is composed of multiple screens, login, all notes screen to display notes, AddNotes screen to add notes, Edit Notes screen to make updates to the already existing notes, and a comment screen where users can have discussion regarding a particular note. 
GetX is used for managing the state of the application and also managing the business logic of the application whereas Bloc is used to track internet connectivity and inform users about the internet state. Additionally, the application is equipped with proper error handling by using utilities such as snackbar to show the outcome of the user interactions.
